### Usage Guidelines

- **Primary actions**: Use Cyan-600 for CTAs and key interactions
- **Secondary actions**: Use Purple-600 for alternative paths
- **Text**: Use Slate-900 for headings, Slate-600 for body text
- **Backgrounds**: Use White for main content areas
- **Hover states**: Use Cyan-300/Purple-300 for hover effects on buttons

## üîÑ Component Usage Patterns

### Buttons

- **Primary**: Use Cyan-600 background with white text for main actions
- **Secondary**: Use Purple-600 background with white text for alternative actions
- **Disabled**: Use Slate-300 with Slate-600 text for unavailable actions

### Layout

- Use spacing tokens consistently for margins and padding
- Maintain 16px/24px gutters between layout elements
- Consider mobile-first design with responsive breakpoints at 768px and 1440px
